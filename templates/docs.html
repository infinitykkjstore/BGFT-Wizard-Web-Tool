BGFT Wizard API Documentation

Current host: {{ host }}

Quick Notes

  • All endpoints are HTTP GET except cleanup (POST).
  • Response format is JSON (except file download/icon endpoints).
  • User-facing logs are session-isolated via cookie session.
  • Full internal logs remain only in server output and logs/server.log.

Status

GET {{ host }}/api/status

Returns environment readiness.

curl -s "{{ host }}/api/status"

User Logs (Session)

GET {{ host }}/api/logs

Returns logs only for the current browser/client session.

curl -s "{{ host }}/api/logs"

Extract Metadata

GET {{ host }}/api/meta?url=<PKG_OR_MANIFEST_URL>

Reads PKG/manifest metadata: title, ids, type, size, category, and icon URL.

curl -G -s "{{ host }}/api/meta" --data-urlencode "url=https://example.com/game.pkg"

Build Payload

GET {{ host }}/api/build?url=...&name=...&id=...&icon=...&type=...&size=...

Compiles payload and returns output filename for download.

curl -G -s "{{ host }}/api/build" \
  --data-urlencode "url=https://example.com/game.pkg" \
  --data-urlencode "name=My Game" \
  --data-urlencode "id=CUSA00000" \
  --data-urlencode "icon={{ host }}/api/icon/sample.png" \
  --data-urlencode "type=PS4GD" \
  --data-urlencode "size=1234567890"

Download Payload

GET {{ host }}/api/download/<generated-file>

Downloads compiled payload file as payload.bin.

curl -L "{{ host }}/api/download/REPLACE_WITH_FILE.bin" -o payload.bin

Icon Retrieval

GET {{ host }}/api/icon/<filename>

Returns stored icon (PNG) extracted during metadata processing.

curl -L "{{ host }}/api/icon/REPLACE_WITH_ICON.png" -o icon.png

Cleanup Old Payloads

POST {{ host }}/api/cleanup

Deletes old temporary payload files from server temp folder.

curl -X POST -s "{{ host }}/api/cleanup"
